# Amazon Clone

## Project Description

This project is a simple Amazon Clone web application built using HTML, CSS, and JavaScript.
It replicates the basic layout and user interface of the Amazon homepage.

## Features

* Amazon style homepage UI
* Product cards display
* Navigation bar with search option
* Responsive layout for different screen sizes

## Technologies Used

* HTML
* CSS
* JavaScript

## Project Structure

amazon-clone
│
├── index.html
├── style.css
└── images/

## How to Run the Project

1. Download or clone the repository
2. Extract the files if downloaded as ZIP
3. Open `index.html` in your browser
